# Идея: ReAct-агент банковского обслуживания клиентов

## Описание проекта

Автономный ИИ-агент (паттерн ReAct) в виде Telegram-бота для консультирования клиентов по банковским продуктам. Агент сам решает, когда искать информацию в базе знаний, и формирует поисковые запросы без отдельного узла query transformation.

## Основная задача

Бот ведёт диалог с пользователем и отвечает на вопросы о кредитах, вкладах и других услугах Сбербанка:

- **ReAct-цикл**: агент рассуждает → при необходимости вызывает инструмент `rag_search` → формирует ответ на основе найденного контекста и истории диалога
- **Поиск в базе знаний**: PDF и JSON в `data/`; три режима retrieval (`semantic`, `hybrid`, `hybrid+reranker`) — как в Advanced RAG
- **Контекст диалога**: история хранится в `MemorySaver` (thread_id = chat_id)

## Ключевые особенности

- **Платформа**: Telegram (aiogram, polling)
- **Агент**: `create_agent()` из LangChain 1.0 + `MemorySaver`
- **Инструмент `rag_search`**: обёртка над текущим retriever; агент передаёт поисковую фразу; возвращает `{"sources": [...]}` с полным `page_content` (для RAGAS) и метаданными
- **Системный промпт**: инструкции когда вызывать `rag_search`, few-shot примеры, подсказки по формулировке запросов
- **База знаний**: PDF в `data/`; JSON `sberbank_help_documents.json` (при объединённой индексации)
- **Retrieval** (без query transformation — агент сам переформулирует):
  - `semantic` — векторный поиск
  - `hybrid` — Semantic + BM25 (RRF)
  - `hybrid+reranker` — hybrid + Cross-Encoder
- **Мониторинг и качество**: LangSmith-трейсинг, RAGAS evaluation (`/evaluate_dataset`), синтез датасетов (`make dataset`); evaluation извлекает `contexts` из `page_content` документов текущего запроса

## Поведение

- На вопросы по продуктам и условиям банка — агент вызывает `rag_search` и отвечает по найденным документам
- На общие приветствия и уточнения — отвечает из контекста диалога без поиска (если контекста достаточно)
- При `SHOW_SOURCES=true` — в ответе показываются источники из последнего вызова `rag_search`
- При отсутствии информации в документах — честно сообщает об этом
