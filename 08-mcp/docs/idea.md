# Идея: ReAct-агент банковского обслуживания клиентов

## Описание проекта

Автономный ИИ-агент (паттерн ReAct) в виде Telegram-бота для консультирования клиентов по банковским продуктам. Агент сам решает, какой инструмент вызвать: поиск в базе знаний, каталог актуальных продуктов, калькулятор кредита или конвертер валют.

## Основная задача

Бот ведёт диалог с пользователем и отвечает на вопросы о кредитах, вкладах и других услугах Сбербанка:

- **ReAct-цикл**: агент рассуждает → при необходимости вызывает инструмент → формирует ответ на основе результата и истории диалога
- **Поиск в базе знаний** (`rag_search`): PDF и JSON в `data/`; три режима retrieval (`semantic`, `hybrid`, `hybrid+reranker`)
- **Каталог продуктов** (`search_products`, MCP): актуальные ставки, акции, условия — данные, которые меняются динамически и отсутствуют в документах
- **Калькулятор кредита** (`loan_calc`, MCP): расчёт платежа и переплаты через внешний API
- **Конвертер валют** (`currency_convert`): актуальные курсы через ExchangeRate-API
- **Контекст диалога**: история хранится в `MemorySaver` (thread_id = chat_id)

## Ключевые особенности

- **Платформа**: Telegram (aiogram, polling)
- **Агент**: `create_agent()` из LangChain 1.0 + `MemorySaver`
- **Локальные инструменты** (`src/tools.py`): `rag_search`, `currency_convert`
- **MCP-инструменты** (сервер `mcp/mcp-bank-agent`): `search_products`, `loan_calc` — подключаются через `MultiServerMCPClient`; при недоступности MCP-сервера бот работает без них (graceful degradation)
- **Системный промпт**: инструкции когда вызывать каждый инструмент, few-shot примеры, разграничение `rag_search` vs `search_products`
- **База знаний**: PDF в `data/`; JSON `sberbank_help_documents.json` (при объединённой индексации)
- **Каталог продуктов MCP**: `mcp/mcp-bank-agent/data/bank_products.json` — вклады, кредиты, карты, счета с актуальными ставками и акциями
- **Retrieval** (без query transformation — агент сам переформулирует):
  - `semantic` — векторный поиск
  - `hybrid` — Semantic + BM25 (RRF)
  - `hybrid+reranker` — hybrid + Cross-Encoder
- **Мониторинг и качество**: LangSmith-трейсинг, RAGAS evaluation (`/evaluate_dataset`), синтез датасетов (`make dataset`)

## Поведение

- На вопросы о процедурах и правилах из документов — `rag_search`
- На вопросы об актуальных ставках, акциях, сравнении продуктов — `search_products`
- На расчёт платежа по кредиту — `loan_calc` (после получения ставки)
- На конвертацию валют — `currency_convert`
- На приветствия и уточнения — отвечает из контекста диалога без поиска
- При `SHOW_SOURCES=true` — в ответе показываются источники из последнего вызова `rag_search`
- При отсутствии информации — честно сообщает об этом

## Запуск

Для полного набора инструментов нужны два терминала:

1. `make run-mcp-bank` — MCP-сервер на порту 8000
2. `make run` — Telegram-бот

Порядок важен: сначала MCP, потом бот. Бот работает и без MCP благодаря graceful degradation.
